## Антиспам для телеграма

Изначально разрабатывался чтобы решить проблему со спамерами в чатах Вастрик.Клуба, но может использоваться и в чатах других форков, да и просто в любых больших чатах.

### Принцип работы
- Пользователи Клуба с привязанным ботом автоматически добавляются в доверенные
- При эвенте "пользователь вошёл в чат" выдаётся простенькая капча
- Если пользователь уже в доверенных, то все проверки пропускаются
- Если пользователь в известных списках спамеров - нафиг
- Если в сообщении многовато эмодзи - нафиг
- Если в сообщении есть слова, которые маскируются под русские но имеют английские буквы внутри - нафиг. Именно этот аспект сейчас затюнен в основном на русскоязычные чаты и может сработать на нормальные слова из языков которые пишутся кириллицей, но пока что все известные кейсы добавлены в тесты.
- Если в сообщении есть стоп-слова типа "заработок в сети" - нафиг
- Дальше сообщение очищается от эмодзи, пунктуации, диакритиков и скармливается ML. Если ML считает что спам - нафиг.
- Если пользователь написал несколько нормальных сообщений, он добавляется в доверенные - спамеры крайне редко "втираются в доверие" и обычно выдают всё первым-вторым сообщением
- Автоматически бот только удаляет сообщения, банятся пользователи только вручную из админки чтобы снизить урон ложнопозитивных срабатываний
- - Кроме блеклиста. Такие банятся сразу, но это можно выключить
- Если кого-то в чате забанил не бот, в админку приходит уведомление, возможно бот пропустил спам и его стоит добавить в датасет

### Команды админки

Команды доступны в админ-чате или лог-чате бота.

Команды, которые вызываются ответом на сообщение:
- `/spam` - добавляет сообщение в датасет спама и список известных плохих сообщений
- `/ham` - добавляет сообщение в датасет не-спама
- `/check` - прогоняет сообщение через простые фильтры и ML-классификатор

Команды без reply:
- `/stat`, `/stats` - статистика по группам
- `/say <user_id|@username> <текст>` - отправить сообщение пользователю от имени бота
- `/suspicious stats` - статистика системы подозрительных пользователей
- `/suspicious list` - список пользователей с включенным AI-детектом
- `/suspicious help` - справка по suspicious-командам
- `/start` - пользовательский старт в личке с ботом

### Конфиг
Создаём чат-админку, запоминаем ID.
Переменные среды:
- DOORMAN_BOT_API: Токен телеграм бота. Обязательно
- DOORMAN_ADMIN_CHAT: ID чата админки. Обязательно
- DOORMAN_CLUB_SERVICE_TOKEN: Сервис токен для Клуба, создаётся тут: https://vas3k.club/apps/. Если переменная не задана, автоматический аппрув людей из Клуба пропускается.
- DOORMAN_CLUB_URL: Базовый URL для форков Клуба, например https://rationalanswer.club/
- DOORMAN_BLACKLIST_AUTOBAN_DISABLE: Установить в true или 1 чтобы бот не банил людей (не из Клуба, не approved, из блеклиста) при заходе сразу
- DOORMAN_LOW_CONFIDENCE_HAM_ENABLE: Установить в true или 1 чтобы отсылать в админку не-спам который имеет низкий конфиденс скор, полезно для чатов где спамеры постоянно проверяют качество ML на прочность
- DOORMAN_CHANNELS_AUTOBAN_DISABLE: Установить в true или 1 чтобы бот не банил сообщения от имени каналов (кроме тех, которые привязаны к текущей группе)
- DOORMAN_APPROVE_BUTTON: Установить в true или 1 чтобы к каждому удалённому сообщению добавлялась опция "это свой, знакомый, завсегдатай"
- DOORMAN_LOOKALIKE_AUTOBAN_DISABLE: Установить в true или 1 чтобы бот не банил сообщения где есть слова, маскируюиеся под русские

## 🛡️ Как работает защита

Бот использует пошаговый алгоритм проверки новых участников:

### 1. Тихий бан по блэклисту
- Проверка новых участников через архив спамеров [lols.bot](https://lols.bot/)
- Автоматическая блокировка на 4 часа при обнаружении в списке
- Блокировка пользователей с именами длиннее 75 символов
- Всё происходит в тихом режиме без уведомлений

### 2. Капча при входе
- Простая капча с эмодзи для новых участников
- 60 секунд на решение
- Временный бан на 20 минут при неудаче
- Приветственное сообщение после успешного прохождения

### 3. Умный антиспам
- Анализ первых сообщений пользователя
- Поиск спам-паттернов и маскировки через похожие символы
- Детект рекламы и подозрительных ссылок
- Удаление сообщений с перебором эмодзи
- Три сценария: пропуск, спорное сообщение, удаление

### 4. LLM анализ
- Анализ профиля и фото с помощью Gemini 2.5 Flash
- Автоматическое определение подозрительных аккаунтов
- Read-only режим на 10 минут для спорных случаев

### 5. Система доверия
- Пользователь попадает в доверенные после трёх обычных сообщений
- Доверенные пользователи не проходят анализ сообщений

### 6. Каскадный режим
- при проходе сообщений через фильтры, бот может запросить финальное решение о автоматическом действии у AI, что позволяет сократить ручные действия в админке бота.

## 🎯 Команды админки

Команды доступны в админ-чате или лог-чате бота.

Команды ответом на сообщение:
- `/spam` - добавить сообщение в датасет спама и список известных плохих сообщений
- `/ham` - добавить сообщение в датасет не-спама
- `/check` - проверить сообщение через простые фильтры и ML-классификатор

Команды без reply:
- `/stat`, `/stats` - статистика по группам
- `/say <user_id|@username> <текст>` - отправить сообщение пользователю от имени бота
- `/suspicious stats` - статистика системы подозрительных пользователей
- `/suspicious list` - список пользователей с включенным AI-детектом
- `/suspicious help` - справка по suspicious-командам
- `/start` - пользовательский старт в личке с ботом

Inline-действия в админских уведомлениях:
- `approve` / `ban` - одобрить или забанить пользователя из уведомления
- `logban` - бан из лог-чата
- `banprofile` - бан по результатам AI-анализа профиля
- `aiOk` - отметить профиль как безопасный
- `suspicious approve` / `suspicious ban` / `suspicious ai` - действия по подозрительному пользователю из кнопок уведомления

Не реализовано как текстовые команды на данный момент:
- `/suspicious ai <on|off> <userId> <chatId>`
- `/suspicious approve <userId> <chatId>`
- `/suspicious ban <userId> <chatId>`
- `/suspicious cleanup <userId> <chatId>`
- `/admin`, `/stop`, `/ban`

## 🧪 Опыт использования

В первый день бот может показаться строгим к новичкам, но это временно. После написания 3 обычных сообщений пользователи попадают в доверенные, и фильтрация прекращается.

**Важно понимать:**
- Бот **не банит** пользователей за сообщения, только удаляет подозрительные
- Ложные срабатывания отслеживаются и исправляются
- Алгоритмы регулярно обновляются
- Автоматически банятся только пользователи из блэклистов
