# 🚫 Автоматический бан за вход через папки

## Описание

Функция автоматического бана пользователей, которые входят в чат через папки Telegram. Это дополнительная мера защиты от спамеров и нежелательных пользователей.

## Как это работает

Когда пользователь присоединяется к чату через папку (используя `via_chat_folder_invite_link`), бот автоматически:

1. **Проверяет настройку** - включена ли функция в конфигурации
2. **Определяет тип входа** - действительно ли пользователь вошел через папку
3. **Банит пользователя** - удаляет из чата с причиной "Автоматический бан за вход через папку"
4. **Отправляет уведомление** - администратору о совершенном действии

## Настройка

### Включение/отключение

Добавьте в файл `.env`:

```env
# Автоматический бан пользователей, входящих через папки (true/false)
DOORMAN_BAN_FOLDER_INVITE_USERS=false
```

- `true` - функция включена, все пользователи, входящие через папки, будут забанены
- `false` - функция отключена (по умолчанию)

## Логирование

При срабатывании функции в логах появляются записи:

```
🚫 ПОЛЬЗОВАТЕЛЬ ВОШЕЛ ЧЕРЕЗ ПАПКУ: Иван Иванов (id=12345, username=ivan) в группу 'Мой чат' (id=67890)
✅ Пользователь Иван Иванов (id=12345) забанен за вход через папку в группе 'Мой чат'
```

## Уведомления

Администратор получает уведомление с типом `AutoBan` и данными:
- **BanType**: "Вход через папку"
- **Reason**: "Автоматический бан за вход через папку"

## Безопасность

⚠️ **Внимание**: Эта функция может заблокировать легитимных пользователей, если они используют папки для организации чатов. Используйте с осторожностью.

```env

```

### Отключение (по умолчанию):
